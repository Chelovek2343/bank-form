# Bank Form — Многошаговая форма

Форма открытия банковского счёта онлайн с валидацией и прогресс-баром.

## Стек
HTML, CSS, JavaScript (без библиотек)

## Фичи
- 3 шага: личные данные → документы → подтверждение
- Валидация каждого поля на чистом JS
- Ошибки исчезают при вводе
- Прогресс-бар с анимацией
- Финальный экран подтверждения
- Адаптивная верстка
